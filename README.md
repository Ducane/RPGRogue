# Synopsis
This is a RPG empathizing Rogue. please see:  
https://de.wikipedia.org/wiki/Rogue-like

# Tiles
At the moment there are four tiles called 'door', 'floor', 'granite', and 'wall'.

# Combat System
The Combat System is inherited from Rogue.

# World Generation
 - rooms are generated
 - maze is generated
 - items are placed
 - stairs are placed

# Controls
 - Movement: WASD
